# leaf-disease-detection

# 🌿 Detect Leaf Diseases using Deep Learning

## 📌 Project Description

This project is a deep learning-based system designed to detect plant leaf diseases from images using Convolutional Neural Networks (CNN). The system helps in early identification of plant diseases, enabling timely intervention and supporting smart agriculture practices.

---

## 🚀 Features

* 🌱 Image-based plant disease detection
* 🤖 Deep learning model using CNN (TensorFlow)
* 🔍 Image preprocessing and data augmentation
* 📊 Visualization of results using Matplotlib & Seaborn
* ⚡ Efficient and accurate classification system

---

## 🛠️ Technologies Used

* Python
* TensorFlow
* Scikit-learn
* NumPy, Pandas
* Matplotlib, Seaborn
* Streamlit (for deployment)

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/leaf-disease-detection.git
```

2. Navigate to the project folder:

```bash
cd leaf-disease-detection
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run app.py
```
